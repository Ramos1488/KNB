# KNB - Социальная сеть

Полнофункциональная социальная сеть с системой верификации, личными сообщениями и редактированием профиля.

## Функции
- 📝 Создание постов с медиа
- 👥 Система друзей
- ✅ Верификация аккаунтов
- 💬 Личные сообщения
- 🎨 Редактирование профиля
  
## Технологии
- HTML5, CSS3, JavaScript
- LocalStorage для хранения данных
- Font Awesome иконки
- Адаптивный дизайн
